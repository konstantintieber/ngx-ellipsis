lx-ellipsis demo

Features

  • Does not show "Show more" button if content has enough space to be shown without ellipsis.
  • Hides "Show less" button in case the client size grows so much after pressing "Show more" before, that the content can be displayed without ellipsis in one line again.
  • Uses the ResizeObserver API to detect size changes of the span element that contains the content. Resize events are debounced with 500ms by default to reduce change detection runs. This can be configured using the LX_ELLIPSIS_DEBOUNCE_ON_RESIZE injection token.
